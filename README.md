WerdCounter
===========

![word](./static/wordtoyourmotha.jpg)

Based on the exercism.io exercise.

### Run the specs
```
$ mix test
```

### Using iex
```
$ iex -S mix
```

### Process the static file
```elixir
$ WerdCounter.process_file "./static/war_and_peace.txt"
```

or

```elixir
$ WerdCounter.count_words_in_file "./static/war_and_peace.txt"
```
